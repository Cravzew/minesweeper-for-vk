# Сапёр

Проект сделан с помощью [Create React App](https://github.com/facebook/create-react-app) с шаблоном TypeScript.

## Команды

### `npm install`

Установка пакетов \
Без него не заработает

### `npm start`

Запуск приложения \
Открыть [http://localhost:3000](http://localhost:3000) для просмотра в браузере


### `npm run build`

Сборка приложения
